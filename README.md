# MTech MSP Website

Static website for MTech MSP, built to run directly on GitHub Pages.

## Files

- `index.html` - main website page
- `styles.css` - responsive layout and brand styling
- `script.js` - mobile navigation behavior
- `assets/` - logo files used by the site

## GitHub Pages

1. Push these files to a GitHub repository.
2. In GitHub, open `Settings > Pages`.
3. Set the source to the main branch and root folder.
4. Save, then open the GitHub Pages URL GitHub provides.

The site does not require a build step.
